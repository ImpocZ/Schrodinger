## Falling Objects Game
# Padající Objekty
**Tato hra spočívá v tom, že hráč ovládá červený balón, který se pohybuje po x rovině a musí chytit padající barevné balóny, pokud chce přežít**
**Hráč má celkem 5 životů. Když nechytí jeden balón tak o jeden přijde. Pokud mu nezbývá žádný, tak umírá a hra končí.**
**Hráč se postupně zvětšuje při kanibalismu ostatních balónů, které se nemůžou bránit. Po určité době na něj začínají balóny působit jako projímadlo, a tak hráč začne rapidně ztrácet na váze a postupně zrychlí**
*Hra končí, když hráč nazbírá skóre o hodnotě 800. Proč? Protože nemá právo ztrácet čas nad takovou hloupostí.*
